# diy-midi-controller
This project is a DIY USB MIDI controller inspired by the YouTuber, https://www.youtube.com/watch?v=DS3cwTbc7yc

## Goal
Recreate the MIDI controller project demonstrated in this video: [YouTube Link](https://www.youtube.com/watch?v=DS3cwTbc7yc)

## Planned Components
- ESP32 or Arduino
- Sensors / keys / piezos
- LED feedback
- Breadboard or prototyping board

## References
- Original video: https://www.youtube.com/watch?v=DS3cwTbc7yc

## Next Steps
- Prototype the controller on a breadboard
